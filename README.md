# MyProect
Привет, эта игра находиться в стадии разроботки, и ты можешь отлеживать мой прогрес по создании ее.
Чтобы ее запустить тебе нужно:
1)Скачать Unity (желательно версия 2018.3.4f1 и выше).
2)Скачать Zip фай с GitHub
3)Разархивировать его, и сохранить желательно на рабочий стол.
4)Открыть Unity, и нажать вверхнем правом углу на Open и выбрать скаченую папку с GitHub.
5)Когда откроеться Unity нажимаешь на кномку Play вверху в центре и начинаешь играть.
